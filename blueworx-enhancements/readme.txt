Starter plugin scaffold for BlueWorx enhancements.

== Description ==

BlueWorx Enhancements is a baseline WordPress plugin scaffold used to add project-specific functionality.

== Installation ==

1. Upload the `blueworx-enhancements` folder to `/wp-content/plugins/`.
2. Activate the plugin through the `Plugins` screen in WordPress.
